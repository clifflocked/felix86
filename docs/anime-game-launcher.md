# Running anime-game-launcher on felix86

"An Anime Game Launcher" is an alternative launcher for a popular gacha game.

## Step 0 - Make sure you have enough space
The installed game takes about 190 GBs of space.

## Step 1 - Download
Download the latest release from [https://github.com/an-anime-team/an-anime-game-launcher/](https://github.com/an-anime-team/an-anime-game-launcher/).

## Step 2 - Install dependencies
Inside the emulated bash environment (run `felix86 --shell`), install the dependencies `git p7zip-full webp` using your package manager.

For an Ubuntu rootfs, use:
```
sudo apt update
sudo apt install git p7zip-full webp
```

## Step 3 - Install
Go through the anime-game-launcher installer to install Wine and DXVK.

Once it's finished, install the game.
